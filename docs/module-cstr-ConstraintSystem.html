Class: ConstraintSystem

(protected) cstr~ConstraintSystem()

Class for system-wide behavior. Variables whose formulas are currently demanding the value of another variable are found on the demandingVariables stack during the sweep phase of the lazy mark-and-sweep dependency management. Daemons in the daemon queue are executed prior to sweeping out-of-date formulas when getting the current value of a variable.
