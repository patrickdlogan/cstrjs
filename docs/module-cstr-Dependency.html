Class: Dependency

(protected) cstr~Dependency(variablenon-null)

Class for maintaining a reference from one variable to a second variable where the formula of the second variable depends on the value of the first. The dependency captures the "tick" as of when the dependency is established. Comparing the current tick of the second variable to the tick of the Dependency instance determines whether or not the dependency is out of date.
